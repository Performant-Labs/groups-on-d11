do_streams — shared stream shell wireframe (issue #109)

Low-fidelity mockup of the theme hook do_streams_shell (do-streams-shell.html.twig). Covers scope tabs, ranking control, and the shell wrapping existing stream_card-rendered rows, across 6 states. This is a visual mockup only — no real routes, Twig logic, or PHP are wired.

Parameterization note (acceptance criterion: no hardcoded routes). Every tab and ranking control below is annotated with its preprocess-variable origin (scope_tabs[n].id / ranking_control[n].id) rather than a literal href. In the real template, each control's link/action target is built from the url_or_param value the preprocess function supplies — never a hardcoded path string in the Twig file. The wireframe below labels controls with their logical id to make this explicit; it does not print any URL.
